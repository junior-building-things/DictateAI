Download for Apple Silicon

How to install:

  1. Download and drag DictateAI into Applications.
  2. Run this command in the terminal: 
    xattr -dr com.apple.quarantine /Applications/DictateAI.app

*This is to remove macOS's quarantine flag for non-notarized builds.

How to use:

  1. Enable microphone and accessibility permissions.
  2. Insert your API keys.
  3. Enjoy~
