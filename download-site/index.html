Your voice,
Perfectly written.

Turn your voice into clean, ready-to-send writing with full control over model selection and prompt customization.

Download for Apple Silicon

Install

  1. Download the DMG and drag DictateAI into Applications.
  2. Allow Microphone access.
  3. Allow Accessibility access (to enable auto-paste).

How to use

  1. Press and hold the hotkey (e.g. Command+A).
  2. Speak.
  3. Release.

Security Note

This is a direct macOS build. It is not notarized yet, so macOS may show a security warning on first launch.

If macOS blocks opening the app

Move the app to Applications first, then right-click it and choose Open. If Gatekeeper still blocks it, run: 

xattr -dr com.apple.quarantine /Applications/DictateAI.app
